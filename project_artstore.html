CREATED WITH :

DESCRIPTION

A web application of an online store selling art supplies, built dynamically with Node.js and MySQL. It features CRUD operations allowing users to browse, manage, and purchase art supplies with ease. Each product is organized with a user friendly interface providing a seamless shopping experience.

FEATURES

Display products by category, individual item, or view all.

Search for products

Add products to cart

Update quantity of products in cart

Delete products individually

Delete All products in cart after purchase

Display dynamic message in conatct page when form is submited
