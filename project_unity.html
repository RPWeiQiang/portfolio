DIAMOND HESIT

CATERGORY :

  • Game Design & Development
  • VR

CREATED WITH :

  • Unity
  • C++

DESCRIPTION

A immersive VR escape room that with a heist theme with reference to heist scenarios where someone does a crime a crime in which valuable things are taken illegally and often violently from a place or person by any means a person could think of. The objective of the heist chosen was a big perious dimond.There was some inspiration being drawn would be base of heist movies like oceans 8 as is more of an exaggerated form of a heist to make it more interesting. The VR escape room game was created using unity engine to create the enviromnet of the game in 3D while using programming langusge C++ to control how the game would flow and mechaniques/behaviors of the game.

GAME FLOW

1.

To get into the vault, find out password and then turning the vault door turning it three times to enter.

2.

Go to the where the diamond is stored and break the display case with a nearby gold block to get to the diamond.

3.

Alarm would sound and red would turn red and the vault would be closed, trapping the player.

4.

Rush to the vault door where the power box is and find the key hidden in the room to open it up.

5.

Connect the wire in the power box and vault door would open.

6.

Rush to the ladder and make your escape! Don’t forget to put diamond in the bag too. END
